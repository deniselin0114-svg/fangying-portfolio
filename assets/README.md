# Portfolio assets

Place the About-me portrait photo here as **`fangying.jpg`**.

The intro section in [`../index.html`](../index.html) references it via:

```html
<img class="intro__photo" src="assets/fangying.jpg" alt="Portrait of Fangying Lin" />
```

To swap the photo, replace `fangying.jpg` (keep the same filename), or update the
`src` in `index.html` and the `.intro__photo` rules in `../styles.css` if you use
a different name or aspect ratio.
